# Sentiment-Analysis
Sentiment Analysis 

This project builds a machine learning model to classify tweets as Positive or Negative using the Sentiment140 dataset.

 About

Cleaned and preprocessed tweet text
Converted text into numerical features using TF-IDF
Trained models like Logistic Regression
Evaluated performance using accuracy and confusion matrix

 Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib

 Result
Achieved around 80% accuracy on test data.
